# 1. 總則說明

## 1.1 目的
為突破目前汽車產業軟體從業人員不足與封閉的開發環境，「FoxtronPi」平台以靈活強大的軟體支援，提供車用軟體開發的應用平台。它採用 Diagnostic communication over Internet Protocol (DoIP) 技術，進行車輛平台動態控制。

## 1.2 安全需知
FoxtronPi 為提供外部控制器操作車輛的平台，涉及車輛動態，為確保人車安全，請務必遵循下列規範:
1. **測試場域**: 僅能在空曠且封閉之場域進行測試，時速不得高於 30km/h。
2. **法規遵循**: 不得有任何違法或超速行為，並遵守在地相關法規。
3. **文件熟悉**: 操作前請完整閱讀並熟悉本文件之流程與限制條件。
4. **初次調參建議**: 初次使用或參數變更請由低速開始 (建議 5-10km)，逐步提升。
5. **人員配置**: 需採雙人作業 (安全駕駛＋測試員)，安全駕駛須具備相應駕照。
6. **緊急停止**: 車輛配置紅色緊急按鈕，可執行緊急停止機制。
7. **車況檢查**: 測試前完成制動、轉向、輪胎、電量與 DTC 狀態檢查。
8. **環境條件**: 避免於雨雪、低附著或視線不良環境操作。
9. **權限與保護**: 遵循寫入保護、解除流程，禁止未授權人員修改控制參數。
10. **回復機制**: 每次測試結束，請復位控制，確認車輛回到安全靜止狀態後方可離場。

## 1.3 聲明
本公司提供的「捐贈或借用車輛」係基於捐贈或借用目的，不涉及買賣或商業交易行為，且不提供任何形式的保證。
