# 9. 紅色緊急按鈕操作方法

在緊急情況下，請正確操作紅色緊急按鈕以確保安全。

## 9.1 緊急停止按鈕 (紅色) 使用說明
僅限緊急情況使用。一旦按下，系統將執行下列操作：
- **切斷高壓主電**: 車輛將斷開動力電源。
- **功能中止**: 轉向/煞車輔助功能可能受影響。
- **DTC 記錄**: 各車載 ECU 會記錄「由急停導致的異常」之 DTC。

## 9.2 何時可以按
1. 發生人員傷害風險。
2. 設備或車輛失控。
3. 起火冒煙或其他立即危害情況。
4. 測試中出現不可控狀態。

## 9.3 按下後的影響
- 高壓電切斷，動力中止。
- 儀表會亮起嚴重故障警示，IV I會出現故障警告。
- 請按下立即關閉電源。

## 9.4 復歸標準操作 (SOP)
1. **順時針旋轉**: 紅色按鈕復歸。
2. **小電瓶負極接上**: 若之前曾卸下，請重新接上 (需等待 30 秒)。
3. **啟動車輛**: 踩下煞車，將 Key 靠近方向盤感應區域，按下啟動。
4. **檢查狀態**: 確認儀表無異常，啟動成功。
