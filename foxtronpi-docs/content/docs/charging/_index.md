# 5. 車輛充電與電源供應

## 充電方式
FoxtronPi 車型目前僅支援慢充 (AC 充電)，不支援快充 (DC 充電)。請勿使用快充設備，以避免影響充電系統運作，請使用符合 Type 2 的充電接頭進行充電。

## 電源供應
由於FoxtronPi車型並未配備對外供電的AC 110V插座，因此在電源供應方面，可透過車內12V點菸器接上電源轉換器，以提供AC 110V電源。根據FoxtronPi車輛的配置，其12V點菸器保險絲規格為15A，因此最大可支援約180W以下的電力輸出。若所使用的運算電腦或設備所需功率超過180W，則須另行架設專用電源，無法依賴車輛內建之12V點菸器進行供電。

外部設備運作功率
- 小於 180W：12V點菸器接口（透過12V outlet to 110V變壓器）
- 大於 180W：不適用
