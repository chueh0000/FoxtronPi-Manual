    6. 控制訊號操作流程與限制條件#

    在進行控制操作前，請務必了解操作流程與安全限制。

    6.1 控制啟用流程#

    1. 將 Ctrl_Enable_Switch 設為 1 (啟用)。
    2. 若設定失敗，所有控制命令將會被拒絕。

    6.2 車輛動態操作流程#

    1. 加減速度控制 (6.2.1):
      • 限制: -15 to 1.5 m/s²。
      • 建議: 初次測試建議以 0.05 m/s² 逐步遞增。
      • 條件: 控制前需拉起 Acc_A 驗證旗標。
    2. 目標速度控制 (6.2.2):
      • 限制: 最大 170 km/h。
      • 建議: 初次實驗建議設定在 5-20 km/h 區間。
    3. 方向盤角度控制 (6.2.3):
      • 範圍: ±360 deg。
      • 限制: 方向盤轉速不可超過 500 deg/s。
    4. 方向盤扭矩控制 (6.2.4):
      • 範圍: -3 to 3 Nm。
    5. 檔位控制 (6.2.5):
      • 條件: 車輛需靜止且 P 檔。
      • 檔位定義: P=2, R=7, N=4, D=5。

    6.3 燈光控制操作流程#

    每一項燈光控制 (位置燈、近光燈、遠光燈等) 都有對應的 Enable 旗標與控制命令。需先啟用該燈光的控制旗標，方可對該燈光進行操作。

    操作建議#

    • 避免同時開啟衝突組合: 如位置燈、近光燈、遠光燈與右晝行燈不可同時開啟。
    • 重啟保險: 一旦切回 P 檔，系統會重新檢查安全狀態，控制流程可能需要重新執行。
