    3. 車輛規格#

    3.1 FoxtronPi整車規格#

    • 車型: D31L24 (衍生車)
    • 尺寸: 4695 * 1895 * 1625 mm
    • 軸距: 2920 mm
    • 車重: 2130 kg (5人 H/L RWD)
    • 電機: 交流永磁同步馬達 (172 kW, 340 Nm)
    • 電池: LFP (57.73 / 82.74 kWh)
    • 續航: 505 / 700 km
    • 快充: <30 mins (10%-80%)

    3.2 硬體平台動態控制規格#

    3.2.1 煞車控制規格#

    • 類型: 液壓式制動
    • 性能: 1.07G 最大減速度 (ABS 啟動)
    • 響應時間: < 100 ms

    3.2.2 轉向控制規格#

    • 類型: 前軸轉向, 雙小齒輪 EPS
    • 性能: 啟動響應 < 150 ms, 角度誤差 < 0.5 deg

    3.2.3 動力控制規格#

    • 性能: 速度 <= 40 km/h, 輪端扭力 3900 Nm

    3.2.1 FoxtronPi 通訊規格#

    • 協定: UDS / DoIP
    • 性能: 每筆請求約 40ms 延遲
