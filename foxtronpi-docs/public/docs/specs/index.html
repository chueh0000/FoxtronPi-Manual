    3. 車輛規格#

    3.1 FoxtronPi 整車規格#

    • 車型：D31L24 (衍生車)
    • 尺寸及重量
      • 車長*車寬*車高：4695 * 1895 * 1625 mm
      • 軸距：2920 mm
      • 前軸輪距/後軸輪距：1639 / 1635 mm
      • 車重：2130 kg (5人 H/L RWD)
    • 電機及電池
      • 電機類型：交流永磁同步馬達
      • 馬達控制器：540A / 480V
      • 最大扭矩：340Nm @ 350V
      • 最大功率：172kw @ 350V
      • 變速箱齒比：11.00582
      • 最高轉速：5500 rpm
      • 電池類型：LFP
      • 電池模組：144.33 / 206.85 Ah
      • 續航里程：505 / 700 km
      • 電池能量：57.73 / 82.74 kWh
      • 快充時間 10%~80%：< 30mins
      • 慢充 (0%~90%) / (0~100%)：10 / 14 hours
    • 懸吊、煞車、轉向
      • 驅動型式：RWD (後輪驅動)
      • 前懸吊型式：麥佛遜懸吊附防傾桿
      • 後懸吊型式：5 連桿附防傾桿
      • 前後輪煞車型式：碟煞
      • 輪胎規格：225/55R18
      • 最小迴轉半徑：5.8 m
    • 科技
      • B-Paddle (One Paddle Drive)：○
      • PRND 檔位線控系統：○
      • 多媒體中控屏尺寸 (吋)：15.6
      • 抬頭顯示器：X
      • 防鎖死煞車、制動力分配、煞車輔助、動力回收、電子駐車系統：○

    3.2 硬體平台動態控制規格#

    3.2.1 煞車控制規格#

    • 煞車類型：液壓式制動
    • 是否可控制：可控制
    • 系統性能
      • 1.07G 最大減速度 (輪胎鎖死 ABS 啟動)
      • 400 毫秒(ms) 輪端鎖死時間
      • < 100 毫秒(ms) 減速度響應時間
      • ≤ 10% 穩態循跡誤差
    • 接收訊號
      • 10 毫秒(ms)
      • 車輛減速度訊號
    • 回饋訊號
      • 10 毫秒(ms)
      • 四輪輪速
      • 直行/側向加速度及橫擺角速度
      • 煞車系統狀態
      • 緊急煞車訊號 (PBA)

    3.2.2 轉向控制規格#

    • 類型
      • 前軸轉向
      • 雙小齒輪 EPS
    • 是否可控制：可控制
    • 系統性能
      • 啟動響應 < 150 ms
      • 角度控制穩態誤差 < 0.5 deg
    • 接收訊號
      • 訊號週期 10 ms
      • 轉向扭力控制相關訊號
      • 轉向角度控制相關訊號
    • 回饋訊號
      • 訊號週期 10 ms
      • 轉向角度與駕駛扭力
      • 控制狀態
      • ECU錯誤內容

    3.2.3 動力控制規格#

    • 動力控制類型
      • 後驅單電機
      • 單速齒輪 / 開放式差速器
    • 是否可控制：可控制
    • 系統性能
      • ≤ 40 公里/小時
      • 3900Nm 輪端扭力
      • ≤ 50ms 輪端響應時間
      • ≤ 5% 穩態誤差
    • 接收訊號
      • 10 毫秒(ms)
      • 車輛加減速度訊號
    • 回饋訊號
      • 10 毫秒(ms)
      • 馬達端扭力
      • 最大、最小輪端扭力
      • 馬達轉速
      • 12V、高壓電池狀態

    3.2.1 FoxtronPi 通訊規格#

    • 通訊協定：UDS / DoIP
    • 是否可控制：可控制
    • 通訊性能
      • 每次僅可發送一筆控制或讀取請求。需在收到回覆或逾時後，再送出下一筆。
      • 每筆請求約40ms延遲
