    4. 車輛充電與電源供應#

    充電方式#

    FoxtronPi 車型目前僅支援慢充 (AC 充電)，不支援快充 (DC 充電)。請使用符合 Type 2 的充電接頭進行充電。

    電源供應#

    FoxtronPi 車型並未配備對外供電的 AC 110V 插座。在電源供應方面，可透過車內 12V 點菸器接上電源轉換器，以提供 AC 110V 電源。

    • 12V 點菸器保險絲規格: 15A
    • 最大支援功率: 約 180W 以下。若設備功率超過 180W，則須另行架設專用電源，無法依賴車輛內建之 12V 點菸器供電。
